# Svelte Todo List

A todo list application built with Svelte.

## Requirements

- **Node.js**: v16 or higher
- **npm**: v7 or higher

## Setup

1. Clone the repository or navigate to the project directory:
```bash
cd svelte_todoList
```

2. Install dependencies:
```bash
npm install
```

## Run the Application

Start the development server:
```bash
npm run dev
```
